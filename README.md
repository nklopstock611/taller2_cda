# Taller 2 - Ciencia de Datos Aplicada
El repositorio tiene dos archivos principales:
- Notebook con limpieza de datos, revisión de distribuciones de variables y sus correlaciones con la objetivo, entrenamiento y evaluación de modelos, y anális de interpretabilidad usando LIME y SHAPLEY.
- Documento PDF donde se describen y justifican todos los resultados sacados del notebook. Además, la generación de valor del modelo para el negocio del caso dado e insights finales del mismo.

También, está incluido el dataset utilizado (apartamentos.csv).
